# Online Food Delivery Crisis Recovery
A professional project analyzing a large Food delivery dataset using *Microsoft Excel* and *Power BI*. This project provides insights for a crisis recovery to an online food delivery startup.

## 📌 Project Overview
This project focuses on analyzing the impact of business crisis on a startup company using **Excel** for data querying and **Power BI** for visualization. This is a self-initiated real-time analytics project inspired by a business case challenge.
It includes:

- Data cleaning  
- Date formatting  
- Calculated fields  
- Pivot Table analysis
- Power Query data formatting
- Creating measures
- Business insights
- GitHub documentation
  
------
## 🎯 Business Problem

Online delivery startup company experienced a significant crisis during June–September 2025.

The goal of this project is to:

- Measure performance decline during the crisis
- Identify most affected cities and restaurants
- Analyze customer churn and loyalty impact
- Evaluate revenue loss and rating fluctuations

-----
## 🗂 Dataset Description

The dataset includes:

- Order data (date, city, restaurant, order value)
- Customer ratings
- Delivery time & SLA data
- Cancellation details
- Customer review text
- Revenue components (subtotal, discount, delivery fee)

---

## 🔧 Data Preparation Steps

- Formatted date column into proper Excel date type.
- Merged multiple Excel sheets using Power Query editor.

----
## 🛠 Tools & Technologies Used

- **Excel**– Data validation & pivot analysis
- **Power BI**– Dashboard creation & Visualization

----
## 📈 Key Analysis Performed

- Monthly order comparison (Pre-crisis vs Crisis).
- Top 5 cities with highest order decline.
- High-volume restaurants with largest drop.
- Cancellation rate trend analysis.
- Delivery SLA performance comparison.
- Ratings fluctuation analysis.
- Sentiment analysis (Negative keyword frequency).
- Revenue loss estimation.
- Loyalty impact analysis (High-frequency customers).

----

## 📊 Key Insights

- Significant decline in total orders during crisis period.
- Tier-1 cities showed highest revenue loss.
- Delivery delays strongly correlated with rating drops.
- Loyal customers reduced order frequency.
- Increase in cancellation rate during peak outage months.
- Negative keywords increased during SLA failure period.
